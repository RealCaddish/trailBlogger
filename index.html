Greetings and welcome to Trail Blogger!

×

This is a personal project I created to document my hiking experiences. I wanted a place where I could keep track of the trails I explore, upload photos from my adventures, and write down my thoughts while they're still fresh.

It's become my go-to space for preserving those moments on the trail—what I saw, how I felt, and the details I don't want to forget. If you hike too, I hope this helps you capture and remember your own journeys.

How to Use It

  • Click Add Trail to import a GeoJSON trail file
  • Add photos and write about your hike
  • Track which trails you've completed
  • Use the Data button to backup your journal

Setup (One-Time)

To get photos working, you need to run the Python backend:

  1. Open terminal in the project folder
  2. Run: python server.py
  3. Open: http://localhost:5000

That's it! Everything stays on your computer.
